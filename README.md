# 前端学习计划 (30天)

HTML5 + CSS + Vue3 + TypeScript 每日学习记录。

## 仓库说明

30天前端学习计划，涵盖：
- HTML5 基础与进阶
- CSS 基础与进阶  
- Vue3 核心概念与实战
- TypeScript 基础

## 文件结构

```
/ 前置知识/      # HTTP/TCP、浏览器原理、Web 标准
/ HTML5/        # HTML5 标签、语义化、表单等
/ CSS/          # CSS 布局、 Flexbox、 Grid、动画等
/ Vue3/         # 响应式、组件、 Composition API、 Pinia 等
/ TypeScript/   # 类型系统、泛型、接口等
/ 实战项目/      # 阶段实战练习
/ 笔记/          # 每日学习笔记
```

## 学习路线

| 阶段 | 内容 | 天数 |
|------|------|------|
| 1 | 前置知识 + HTML5 | 1-7 |
| 2 | CSS 基础 + 进阶 | 8-14 |
| 3 | Vue3 核心 | 15-21 |
| 4 | TypeScript + Vue3 实战 | 22-28 |
| 5 | 综合项目 + 复习 | 29-30 |
