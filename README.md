# HTML5/CSS Learning

记录 HTML5/CSS 前置知识与每日学习过程。

## 仓库说明

- 前置知识：HTTP/TCP、浏览器工作原理、Web 标准等
- HTML5 基础与进阶
- CSS 基础与进阶
- Web 开发 fundamentals

## 如何使用

1. Clone 到本地：`git clone https://github.com/llh2hxf-lab/html5-css-learning.git`
2. 每日学习后，在对应目录添加笔记
3. 按照文件夹结构整理内容

## 文件结构

```
/ 前置知识/
/ HTML5/
/ CSS/
/ 实践项目/
/ 笔记/          # 每日学习笔记
```

**学习路线**：前置知识 → HTML5 → CSS → 实战项目
